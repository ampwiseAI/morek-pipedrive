# Morek × Pipedrive sales training

Interactive web slideshow for the Morek sales team covering Pipedrive setup, deals, pipelines, activities, and the daily routine. 29 slides, autoplay, 9 languages.

## View it live

Once GitHub Pages is enabled, the deck is live at:
**`https://YOUR-USERNAME.github.io/morek-pipedrive-webinar/`**

The first thing you'll see is a password gate. Password: **`Barcelona2026`**

## Languages

Click the flag in the top-right to switch between English, Eesti, Polski, Čeština, Latviešu, Lietuvių, Français, Português, Suomi.

## Controls

- **Spacebar** — pause / resume autoplay
- **Arrow keys** — manual prev / next
- **F** — full screen
- **M** — open chapter menu
- **Esc** — close menus
- **Number keys 1–9** — jump straight to that slide

## Edit it

Single file: `index.html`. Open it in any code editor. The translations live in the `I18N` object near the bottom of the script tag. The Pipedrive deep-link buttons on the setup slides point to `morek2.pipedrive.com/...` — update those URLs if your instance uses different paths.

## Optional screenshot drop-ins

Two slides will use real Pipedrive screenshots if you drop the files alongside `index.html`:

- `pipedrive-deal-button.png` — for slide 15 (Creating a deal)
- `pipedrive-org-example.png` — for slide 12 (Setting up an organisation)

If the files aren't there, the slides quietly fall back to clean CSS placeholders.
